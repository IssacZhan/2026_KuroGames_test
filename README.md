# 推箱子 + 关卡编辑器

————————————————————

基于Godot 4.6开发
支持多层地图、坑洞、电梯等新机制
包含完整的关卡编辑器

————————————————————

# 经典推箱子玩法
	将箱子移动到所有目标点上、完成所有目标即可过关
# 多层地图
	最多三层、可以通过电梯和坑洞实现纵向移动，扩展谜题维度
# 特殊地形
	电梯：可以瞬间传送到临近楼层的对应位置
	坑洞：可以让箱子和玩家角色掉落，在使用箱子填埋或垫脚后还能直接跨过
# 关卡编辑器
	支持所有游戏中已有元素的绘制
	支持最多3层的编辑
	支持幽灵层显示、辅助玩家编辑
	支持试玩验证，必须通过程序审核和试玩通关才能保存
	支持文件保存，可以在选关界面直接打开游玩
# 内置关卡
	提供了四个能够直接展示机制的基础关卡

————————————————————

操作方式
方向键（↑↓←→）移动，其他所有情况都可以用鼠标和按钮解决
踩上电梯自动传送
掉入坑洞可能死亡或掉落至下层
编辑器模式下、鼠标左键放置物体，工具栏选择类型，楼层切换，试玩按钮验证关卡等

————————————————————

架构
使用单例GameLevelManager、GameSystemManager和CameraManager来管理全局数据
关卡以 JSON 格式存储，编辑器动态导出，游戏运行时即时解析
利用世界坐标偏移并水平排放置各层，摄像机切换聚焦实现切换楼层
支撑移除、连锁掉落、电梯防循环等特殊逻辑都已经过测试并通过
